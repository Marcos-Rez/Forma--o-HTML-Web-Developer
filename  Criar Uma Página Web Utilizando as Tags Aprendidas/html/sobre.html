Desafios de Projeto DIO

Um dos elementos centrais da metodologia educacional da DIO é o conceito de Project-Based Learning / Problem-Based Learning (PBL). Nesse contexto, os Desafios de Projeto (também conhecidos como Labs) têm como objetivo aplicar o PBL em conjunto com o GitHub, para criação de um portfólio de soluções consistente ao longo da jornada educacional.

IMPORTANTE: Os Labs entregues na plataforma da DIO ficam disponíveis para as empressas parceiras avaliá-los tecnicamente.
