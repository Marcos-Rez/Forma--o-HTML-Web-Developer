Sobre o HTML

O html é uma linguagem de marcação de hiper texto usado para criação de páginas web.

Por ser a estrutura basica de uma pag web é imposivel construir uma pagina sem ele dada a sua importancia falarei um pouco sobre ele.

O HTML foi criado na década de 1990, por Tim Berners-Lee, o mesmo criador do WWW (World Wide Web). Inicialmente o HTML foi projetado para interligar instituições de pesquisa próximas, e compartilhar documentos com facilidade.

Pode se dizer que é um conjunto de elementos conectados como testos, imagens, videos e consistem em um conjunto de tags e atributos que são blocos de informções que mostrao ao navegador o inicio e fim de um conteudo.

TAGS

As tags definem o inicio e fim de um elemento mais tambem são uma estrutura semantica que define um sentido ao que esta fazendo parte do seu conteudo.

Atualmente, existem 142 tags HTML disponíveis, porem os navegadores não suportão algumas delas por questões como compatibilidade e outras causas, dentre estas tags temos <html> que define um documento HTML e faz parte das tags que obrigatoriamente formam o documento HTML.

As tags obrigatórias em um docúmento HTML

  • <html> Tag que define uma página html.
  • <head> Contém configurações da página.
  • <body> Cóntem o conteudo que de fato aparece na tela.

As tags mais comuns são

Tags de titulos definem o tamanho do título do documento, sendo <h1> de maior valor hierárquico e <h6> de menor valor hierárquico.

  • <h1>
  • <h2>
  • <h3>

Tags de texto definem inicio efim de um paragrafo, negrito, italico etc.

  • <p>
  • <b>
  • <i>

Tag link utilizada para inserir um link na página, faz a ligação entre páginas.

  • <a>

Tags que inserem midia na página.

  • <img> insere imagens na página.
  • <video> insere video na página.
  • <audio> insere áudio na página.

Tags usadas para criar listas

  • <ol><li>conteudo</li></ol> Conteudo ordenado.
  • <ul><li>conteudo</li></ul> Conteudo não ordenado.

Concluindo

Páginas HTML sao criadas com tags que dão sentido ao documento, por isso quando usamos uma tag semantica definimos não só a estrutura da página, mas tambem seu comportamento. Outra vantagem é a otimização para os mecanismos de buscas (SEO). Quando falamos em desenvolvimento de sites hoje, um dos conhecimentos mais importantes é o SEO, um mecanismo que permite que os endereços apareçam nas primeiras posições do Google e sejam encontrados com facilidade.

O HTML é essencial para a construção de websites. Por essa razão, é interessante se dedicar a ele. Como a demanda por sites é crescente, o aprendizado de HTML é bastante valioso para a carreira em TI e pode ajudar a impulsionar sua vida profissional.
